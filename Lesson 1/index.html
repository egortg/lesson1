Добро Пожаловать в библиотеку тегов

  1. Тег: cite Используется для указания источника цитирования. Отображается курсивом.
  2. Тег: code Представляет фрагмент программного кода, отображается шрифтом семейства monospace.
  3. Тег: del Помечает текст как удаленный, перечёркивая его.
  4. Тег kbd Выделяет текст, который должен быть введён пользователем с клавиатуры, шрифтом семейства monospace.
  5. Тег: ins Выделяет текст подчеркиванием. Применяется для выделения изменений, вносимых в документ.
  6. Тег: mark Выделяет фрагменты текста, помечая их желтым фоном.
  7. Тег: s Отображает текст, не являющийся актуальным, перечеркнутым.
  8. Тег: strong Расставляет акценты в тексте, выделяя полужирным.
  9. Тег: u Выделяет отрывок текста подчёркиванием, без дополнительного акцента.
  10. Тег: table Элемент для создания таблицы.
  12. Тег: br Перенос текста
    на новую строку.
  13. Тег: marquee Я БЕГУ

  14. Тег: h1 Заголовок 1 ур

  15. Тег: h2 Заголовок 2 ур

  16. Тег: h2 Заголовок 3 ур

  17. Тег: h2 Заголовок 4 ур

  18. Тег: h2 Заголовок 5 ур
  19. Тег: hr Горизонтальная линия для тематического разделения параграфов.
  20. Тег: input Создает многофункциональные поля формы, в которые пользователь может вводить данные.
  21. Тег: small Отображает текст шрифтом меньшего размера.
  22. Тег: em Курсив
  23. Тег: pre  Позволяет вывести текст на экран, сохранив изначальное форматирование. Пробелы и переносы строк при этом не удаляются.
  24. Тег: img картинка

  25. Тег: p Определяет параграф.

  26. Тег: q Определяет короткую цитату.
  27. Тег: sup Определяет текст с верхним индексом
  28. Тег: sub Определяет текст с нижним индексом.
  29. Тег: select Определяет раскрывающийся список.
  30. Тег: map Определяет карту-изображение.
